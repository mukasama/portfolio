ABOUT US!

  • This application is for people who love traveling. However, before you go somewhere, It is always a good idea to have background knowledge about the place you are traveling too. Whenever, I meet someone they always ask me questions about where I am from, so this specific application is about my country Uganda as it covers the basic key points about the country and places to visit.

  • Application: This is the application that contains the travel information that the person is looking for. There is a lot in this window such as the historical and cultural information for the country.

  • Music Loop: This is located inside the Application as it plays the number one song in Uganda that people have voted for each week. You can listen to the song as you read about your place of travel.
